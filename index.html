ITAI 2026 · Springer LNNS · SCOPUS · USA

Rule-Based Script Detection and Multilingual Topic Modeling for Low-Resource Bangladeshi Languages

Training-free multilingual NLP framework for a 5-language Bangladesh parallel corpus using Unicode rules, keyword lexicons, and BERTopic-based topic modeling.

✓ Accepted SCOPUS Indexed Springer LNNS First Author
Paper Link →

Research Details

Abstract

This work presents a training-free multilingual NLP pipeline for low-resource Bangladeshi languages, focusing on script detection and topic modeling. The framework combines Unicode-range rules and keyword lexicons for language/script identification, followed by BERTopic with multilingual sentence embeddings to discover conversational topics from a 5-language parallel corpus including English, Bangla, Garo, Marma, and Chakma. The system achieved strong macro-level script detection performance and demonstrated practical usability for low-resource language analysis without requiring large supervised datasets.

ICCAD 2026 · IEEE / IFAC · Lisbon, Portugal

A Leakage-Aware Deep Learning Framework for Multi-Class Classification of Moringa Leaf Diseases

Deep learning framework benchmarking CNN and Vision Transformer architectures for Moringa leaf disease classification with leakage-free grouped splitting.

✓ Accepted IEEE Indexed IFAC First Author
Paper Link →

Research Details

Abstract

This study introduces a leakage-aware deep learning framework for multi-class Moringa leaf disease classification. It benchmarks EfficientNet-B3, ConvNeXt-Tiny, ViT-B/16, Swin V2-Tiny, ResNet50, and MobileNetV3-Large on the MoringaLeafNet dataset. The pipeline applies strict grouped splitting to reduce data leakage and uses HSV-based leaf segmentation to focus model learning on disease-relevant regions. The best model, EfficientNet-B3, achieved strong classification performance with high accuracy, macro F1-score, and ROC-AUC.

B.Sc. Thesis · Target: IEEE Transactions on Medical Imaging

Privacy-Preserving Federated Domain Generalization for Multi-Centre Cardiac Cine MRI Segmentation

F²DG-nnU-Net framework using nnU-Net v2, federated learning, Fourier amplitude mixing, and boundary-weighted consistency for cardiac MRI segmentation.

In Preparation Federated Learning Medical Imaging First Author
Repository →

Research Details

Abstract

This thesis proposes F²DG-nnU-Net, a privacy-preserving federated domain generalization framework for multi-centre cardiac cine MRI segmentation. Built on nnU-Net v2, the method targets LV, RV, and MYO segmentation on the M&Ms dataset under multi-vendor and multi-centre domain shift. It combines client-level low-frequency Fourier amplitude fingerprint sharing, amplitude-mixing based synthetic domain augmentation, and boundary-weighted Jensen–Shannon consistency to improve robustness across unseen vendors and centres without sharing raw images.

Active Research · Target: IEEE Transactions on Medical Imaging

SR-FedCoDA: Federated Continual Domain Adaptation for Medical Image Segmentation

Federated medical imaging framework combining sparse regularization, co-training, and domain adaptation for heterogeneous clinical imaging environments.

In Preparation Federated Learning Domain Adaptation First Author
Repository →

Research Details

Abstract

SR-FedCoDA explores federated continual domain adaptation for medical image segmentation under heterogeneous hospital data distributions. The framework combines sparse regularization to reduce communication overhead and prevent client drift, co-training across federated clients, and domain adaptation strategies for non-IID imaging environments. It is designed for privacy-preserving multi-institutional medical AI where raw clinical data cannot be centralized due to regulatory and ethical constraints.
