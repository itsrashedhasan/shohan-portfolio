## ✨ Features

- ⚡ **Blazing fast** — zero build step, zero JS frameworks
- 🌙 **Dark / Light mode** — system preference detected + manual toggle, persisted in localStorage
- 📱 **Fully responsive** — tested on iPhone SE, Pixel 7, Pixel 7 Pro, tablets, and large monitors
- 🎨 **Glassmorphism UI** — modern glass cards with smooth hover effects
- 🔢 **Animated stats marquee** — scrolling achievement counters
- 🗂️ **Tabbed sections** — Skills, Projects, Research, Experience, Achievements
- 📬 **Working contact form** — powered by Web3Forms, no backend needed
- 🗺️ **Interactive map** — Leaflet.js map in contact section
- 💬 **Bio modal** — "Read Full Story" expands full about section with spring animation
- ⚖️ **Legal modals** — Privacy Policy, Terms of Use, Cookie Policy, Sitemap
- 🔄 **Scroll animations** — AOS (Animate On Scroll) for smooth section reveals
- 📊 **Scroll progress bar** — top-of-page reading indicator
- 🎯 **Active nav highlighting** — nav links update as you scroll

## ⚙️ Run Locally

No build step needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/itsrashedhasan/shohan-portfolio.git

# Open in browser
cd shohan-portfolio
open index.html
```

Or use VS Code Live Server:
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. Opens at `http://localhost:5500`

---

## 🚢 Deploy to GitHub Pages

```bash
git add .
git commit -m "update portfolio"
git push origin main
```

Then in your GitHub repo:
- Go to **Settings → Pages**
- Source: **Deploy from branch → main → / (root)**
- Click **Save**

Live in ~60 seconds at:
```
https://itsrashedhasan.github.io/shohan-portfolio/
```

## 📬 Contact Form Setup

This portfolio uses **Web3Forms** for the contact form — no backend or server required.

1. Sign up free at [web3forms.com](https://web3forms.com)
2. Get your access key
3. Replace the key in `index.html`:
```html
<input type="hidden" name="access_key" value="YOUR_ACCESS_KEY_HERE">
```
